SelectHorizontalScrollItem : MonoBehaviour

描述

接口

字段/属性/事件

类型 名称 读写性（只读，读写） 描述

string description 读写 选项描述

SelectHorizontalScroll selectHorizontalScroll 控制此选项的脚本
